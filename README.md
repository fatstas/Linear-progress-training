# Linear-progress-training
 📖 Как пользоваться
Создание плана тренировок
Установите параметры:

Ваш разовый максимум (кг)

Шаг увеличения веса (кг)

Настройте диапазоны подходов:

Добавьте строки с диапазонами процентов

Укажите схемы подходов (например: "5x5", "3x8")

Сгенерируйте план - программа автоматически рассчитает все тренировки

Пример настройки диапазонов
text
50% - 60%   → 5x10
60% - 70%   → 5x8  
70% - 90%   → 5x5
90% - 100%  → 5x3
Управление пресетами
Сохранение: Введите название и сохраните текущие настройки

Загрузка: Выберите пресет из списка для быстрой загрузки

Удаление: Удалите ненужные пресеты

Калькулятор 1ПМ
Введите рабочий вес и количество повторений

Получите расчеты по 7 различным формулам

Используйте среднее значение для точного определения максимума

💾 Сохранение данных
Программа автоматически сохраняет:

Настройки диапазонов подходов

Созданные пресеты тренировок

Все настройки при закрытии программы

Файлы сохраняются в training_settings.json

🎯 Примеры использования
Для новичков
Старт: 50% от максимума

Финиш: 80% от максимума

Шаг: 2.5 кг

Диапазоны: 50-60% (5x10), 60-70% (5x8), 70-80% (5x5)

Для опытных
Старт: 70% от максимума

Финиш: 100% от максимума

Шаг: 2.5 кг

Диапазоны: 70-80% (5x5), 80-90% (5x3), 90-100% (3x2)

🔧 Технические особенности
Округление весов - до ближайших 2.5 кг

Минимальный вес - 20 кг

Авто-размер окна - адаптация под содержимое

Горячие клавиши - Enter для генерации плана
